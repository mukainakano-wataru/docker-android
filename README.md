# docker-android
This is a docker image to build Android app.

# OS
Ubuntu 16.04

# Installed packages
- Git
- OpenJDK 8
- Android SDK r24.4.1
- Ruby 2.3

# Development

Build and run docker image on your environment with following commands

## Build
```
docker build -t mukky620/docker-android .
```
## Run
```
docker run -it mukky620/docker-android
```
